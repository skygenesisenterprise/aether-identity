# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Aether Identity, please report it to us privately before disclosing it publicly.

### How to Report

**Email**: security@skygenesisenterprise.com
**PGP Key**: Available upon request

Please include:
- Detailed description of the vulnerability
- Steps to reproduce
- Potential impact
- Any proof-of-concept code

### Response Time

We aim to respond within 48 hours and provide a fix within 7 days for critical vulnerabilities.

## Supported Versions

Only the latest version of Aether Identity receives security updates. Please keep your installation up to date.

## Security Best Practices

1. **Keep software updated**
2. **Use strong authentication**
3. **Enable HTTPS**
4. **Regular security audits**
5. **Monitor access logs**

## Security Features

- JWT-based authentication
- Role-based access control (RBAC)
- Multi-factor authentication (MFA)
- Audit logging
- Rate limiting
- HTTPS enforcement
- Secure password hashing

## Responsible Disclosure

We follow responsible disclosure principles:
- Private notification
- Reasonable response time
- Public disclosure after fix
- Credit to researchers

Thank you for helping keep Aether Identity secure!
